电图宝
---
这是一个用于收录B站部分UP视频的电子地图项目，核心目的如下  
- （核心）整理视频，针对不同栏目的视频进行分类展示
- （核心）地图展示，针对地区类栏目，将栏目中的区域在线化 
- （核心）资料收集，针对栏目中所提及的文档、资料、热评进行整理、收录、总结（AI与人工）
- （期望）能够缓存收录的视频
